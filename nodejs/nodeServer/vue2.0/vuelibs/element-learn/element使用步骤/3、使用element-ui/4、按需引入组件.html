由于第三步引入组件引入的是整个element-ui，开发过程中不需要使用所有组件，所以不需要引入所有，所以可以引入一部分组件，根据项目需要引入,操作步骤如下： 1.执行命令 npm install babel-plugin-component -D 安装 babel-plugin-component 2、修改.babelrc 文件，将内容修改为： 将 .babelrc 修改为： { "presets": [["es2015", { "modules": false }]], "plugins": [ [ "component", { "libraryName": "element-ui", "styleLibraryName": "theme-chalk" } ] ] } 3、按需全局引入部分组件： 接下来，如果你只希望引入部分组件，比如 Button 和 Select，那么需要在 main.js 中写入以下内容： import Vue from 'vue'; import { Button, Select } from 'element-ui'; import App from './App.vue'; Vue.component(Button.name, Button); Vue.component(Select.name, Select); /* 或写为 * Vue.use(Button) * Vue.use(Select) */ new Vue({ el: '#app', render: h => h(App) });
