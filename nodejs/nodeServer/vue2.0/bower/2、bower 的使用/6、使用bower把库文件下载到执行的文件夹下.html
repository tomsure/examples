bower 会默认把库文件下载到当前项目根目录的/bower_components下面，但是，一般都不喜欢这样，可以进行配置： 1、在/bower_components中新增一个.bowerrc 文件， 1、使用cmd创建： type null >.bowerrc 这种是在windows 环境下创建 2、touch 命令创建： touch .bowerrc linux 环境下或者windows 的git客户端下 2、配置.bowerrc 文件 在新创建的文件中编写以下内容： { "directory" : "js/feng", //自定义放置库文件的位置 "json" : "", "endpoint" : "", "searchpath" : "", "shorthand_resolver" : "" } 3，同样的cmd命令再执行一遍，这次可以看到文件下载到js/feng中了。
